## 建议尺寸

| 类型 | 建议尺寸 | 说明 |
|---|---|---|
| 评论区主图 / 弹窗图 | `>= 1400 x 1000` | 需要支持点击放大 |
| 评论区普通图片 | `>= 1200 x 900` | 横图优先 |
| 英雄图 / 主图 | `>= 1600 x 900` | 宽幅优先 |
